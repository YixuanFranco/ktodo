# Kotlin TODO [![Build Status][]](https://travis-ci.org/l-lin/ktodo)

> Yet another TODO command line app. This project is used for training purpose

## Getting started

### Prerequisite

- Java 8
- Maven 3

### Installation & Usage

```bash
mvn clean install
java -jar target/ktodo.jar -h
```
